**Description:**
Sorting Algorithm Visualiser is a small web app that helps you understand how different sorting algorithms work. You can enter numbers or generate an array and watch how the elements are sorted step by step. This makes learning sorting algorithms easy and visual.

**Features:**
→ Enter custom input array
→ Generate random array
→ Visual bar representation of elements
→ Shows values below each bar
→ Supports multiple sorting algorithms
→ Real-time sorting visualization

**Algorithms Included:**
→ Bubble Sort
→ Selection Sort
→ Insertion Sort
→ Merge Sort
→ Quick Sort

**Tech Stack:**
→ HTML — Structure of the visualiser
→ CSS — Styling and layout
→ JavaScript — Sorting logic and visualization
